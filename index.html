What is API?

First of all API is short for Application Programming Interface. With an API you can transfer data between webpages or programs.

Your program can, by using an API, display information from another program. The API lets you use features from other webpages. For example displaying a map from Google maps.

An API often comes as a packaged product for you to add to your code. That way you can add URL to your JavaScript code and watch the API synchronize the information on your webpage, and change information as it changes on the original webpage.
